# ilgaz-web

Personal website built with React, TypeScript, and Vite.

**Live:** [ilg.az](https://ilg.az)

## Tech Stack

- React 18
- TypeScript
- Vite
- React Router

## Pages

- `/` - Home
- `/about` - About
- `/blog` - Blog
- `/blog/:slug` - Blog Post
- `/now` - Now
- `/projects` - Projects

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

Deployed on [Netlify](https://netlify.com) with automatic deploys from the `main` branch.
